• ALTER
    • To make changes in an existing table.
  • ALTERADD
    • To add a column
  • Syntax:alter table table_name add column_name definition;
    • To add more then one column
  • Syntax:alter table table_name add (column1 definition,column2 definition);
  • ALTERMODIFY
    • To modify an existing column
  • Syntax:alter table table_name modify column_name definition;
  • ALTERDROP
    • To drop a column
  • Syntax:alter table table_name drop column_name;
  • ALTERRENAME
    • To rename a column
  • Syntax:alter table table_name rename column old_name tonew_name;
  • UPDATESET
    • To update a value in a table.
  • Syntax:update table_nameset column_name=valuewhere condition;
  • DROP
    • To drop(completely delete) the table
  • Syntax:drop table table_name;
  • TRUNCATE
    • To truncate(completely erases data inside) the table
  • Syntax:truncate tabletable_name;
  • DELETE
    • To delete a row or complete data of the table
  • Syntax:delete from table_namewhere condition;
        • delete from table_name; //Same as truncate
  • RENAME
    • To rename a table
  • Syntax:rename old_name tonew_name;
