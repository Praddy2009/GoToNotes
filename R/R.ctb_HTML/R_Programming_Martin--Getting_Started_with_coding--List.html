List


A vector having elements of different type is called list.

How to create a list

x <- list("a" = 2.5, “b” = TRUE, “c” = 1:3)


To examin the structure of list we can use str()

str(x)



In the above example a,b and c are Tags which makes it easier to reference the components of the list.

However we can also create lists without custom tags, then it will come with $



~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

How to access values of list

It can be accessed via index value and tag names




However, [ will give us sublist and not the value to get the value we need to use [[


In place of [[ we can also use $

x$c #is same as x[["c"]]

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

How to modify a list
We can change components of a list through reassignment.



~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

How to add to list
Adding new components is easy. We simply assign values using new tags and it will pop into action.
