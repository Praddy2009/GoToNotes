SQL Statements :
  • Data definition Language
Defines the structure that make up a database.
(CREATE, ALTER, DROP, RENAME, TRUNCATE)
  • Data Manipulation Language
Modifies the content of the tables.
(INSERT, UPDATE, DELETE)
  • Query Statement
Retrives data from the database.
(SELECT)
  • Transaction Control Language
Permanently records the changes made to tables or undoes those changes.
(COMMIT, ROLLBACK, SAVEPOINT)
  • Data Control Language
Gives and removes permission on database.
(GRANT AND REVOKE)

Note: Naming Convention
  • The table and column names can be upto 30 characters long.
  • Table or column name must begin with a letter
  • Names are not case sensitive
  • Spaces and hyphens are not allowed in a table or a column name
  • $, _ and # are allowed
