XML External Entities(XXE)
  • Applications those parse XML Input
  • XML input containing a eference to an external entity is processed by a weakly configured XML parser
  • XXE may lead to disclosure of sensitive data, SSRF, Port Scanning,DOS or RCE

Why it Happens
  • Incorrect processing on XML input and parsing

What can be achieved by it
  • XXE may lead to disclosure of
  • Sensitive data (etc/passwd)
  • SSRF(hit a request to your collaborator/server)
  • Port Scanning(enumerate 127.0.0.1 and ports)
  • Denial Of Service(Denial of Service)
  • RCE (Remote code execution OOB)

How do we fix SDE issues
  • Validate or escape tainted data within the system identifier in the DTD
  • The safest way to prevent XXE is always to disable DTDs (External Entities) completely
  • Depending on the parser, disable DTD's
