2. Create two integer vectors of random integers which are chosen without replacement from the
integers 0, 1, ..... , 999. Both vectors have length 100. Suppose x = (x1, x2, ..... , xn) denotes the
vector xVec and y = (y1, y2, ...... , yn) denotes the vector yVec.

a. Compute:
sin(y1)/cos(x1)+ sin(y2)/cos(x2)+ .............. + sin(yn)/cos(xn)

b. Pick out the values in yVec which are > 600.

c. Find index positions in yVec for values which are > 600.

d. What are the values in xVec which are same in yVec and also at same position


Program:

xVec <- sample(0:999,100)
yVec <- sample(0:999,100)

#a
a=0
for( n in 1:100)
{
a=a+sin(yVec[n])/cos(xVec[n])
}
print(a)


#b
for( s in yVec)
{
if(s>600)
{
print(s)
}
}

#c
for( p in 1:100)
{
if(yVec[p]>600)
{
print(p)
}
}

#d
for( e in 1:100)
{
if(xVec[e]==yVec[e])
{
print(e)
}
}
