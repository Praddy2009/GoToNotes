New terms
API: Application programming Interface
Using API a user can ask a server(API provider) to do some service(run program stored on server) and give back the output to the user without logging in to server.

For ex: We move to a app which takesin image and gives back refining it.
Or let say when we google we ask google server to please search our keyword and give back the response

CGI: Common Gateway Interface
It's used to run/execute a webapp on webserver.

OSI Model : Open system Interconnection
Networking related stuff
