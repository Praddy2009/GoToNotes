Now, let’s see how we can add an animation into the frame you’ve created. 

In Spark AR, animations need to be added as an image sequence or a GIF.
    >In the Assets panel, click on Add Asset > Animation Sequence
  • >Type in a name for your animation sequence - for example 
    anim1
     - and then in the Inspector panel click on the Choose File next to the Texture option
  • >Navigate to the folder you downloaded, open the folder called Seq.
  • >Select all the images here and then import them into Spark AR.
    >
Now, click on Add Asset and select Material
  • >In the inspector panel of animated material in texture add animation sequence
  • >Add rectangle and assign the material


Note: You can use Layers panel to create layer(top, mid , lowest) and assign it to canvas elements
