Using component with known Vulnerabilities
  • If you do not know the version of all components you use,this may lead to server issues.
  • Vulnerable System,Application,Software,Protocol running leads to these type of attack.
  • Outdated Softwares,Zero Days vulnerabilities

Why it Happens
  • Old softwares running which are outdated or contains vulns
  • Insecure/Obselete features enabled or installed
  • Eg: Vulnerable version of Dropbear SSHRunning leads to SSH takeover
  • Eg: Vulnerable version of Win 7 will lead to Double Pulsar exploitation.

What can be achieved by it
  • RCE
  • Malware Injections
  • Server Injection
  • Complete takeover of the Server

How do we fix SM issues
  • Never keep the default configuration and go live
  • Update all software and applications
  • Disable all unnecessary Functions
