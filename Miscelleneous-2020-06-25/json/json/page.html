JSON: Javascript Object Notation

Its an open standard file formatthat uses human readable text to transmit data objects consisting of attribute--value pairs & array data types

Its a common data format used for asynchronous browser-server communication(https://www.quora.com/What-are-asynchronous-web-applications), including as replacement for XML in some AJAX-style systems.

Uses the extension .json




It has two parts namely:

KEY & VALUE



Note: Key is always String type
  • The value is stored at key
