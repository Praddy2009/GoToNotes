Data
It can be defined as a representation of facts, concepts, or instructions in a formalized manner. It's generally raw & processed.

Data becomes information when its processed

Information
It is organized or classified data,which has some meaningful values for the receiver. It's a processed data on which decisions & actions are based.

Drawbacks of file system
  • Data isolation: Because data are scattered in various files, and files may be in different formats, writing new application to retrieve the appropriate data is difficult.
  • Data duplication: Redundant data.
  • Dependncy on application programs: Changing files would lead to change in application program.

Database Management System
It is a collection of interrelated data and set of programs to store & access those data in an easy and effective manner .


Need of DBMS
  • Storage: Since redundency of data is eliminated therefore data takes less space.
  • Fast retrieval of data: the DBMS program makes sure the data can be retrieved as quickly as possible.


Types of DBMS
  • Depending on number of users:
    • Single user database system:Supports only one user at a time.
    • Multi user database system:Supports multiple users user at the same time.Small group(workgroup DB) Large scale(Enterprise DB)
  • Depending on location of DB:
    • Centralized database system: It supports data located at single site or single place.
    • Distributed database system: It supports data distributed across several different sites.


Drawbacks of DBMS
  • Complexity Increased
  • Requirement of new and specialized manpower.
.
  • Large size of DBMS
  • Implimentation cost is high
  • Performance: Making DBMS program suitable for various application affects it's performance.
